Module 23 — macOS Dock Navigation

Icons magnify as you hover across

A navigation dock at the bottom of the screen where icons scale up as the cursor approaches. Hover across the dock below.

Why it works

The dock magnification gives the active item visual priority while keeping all options visible. It's playful, space-efficient, and immediately familiar to anyone who's used a Mac. Works for app navigation, tool palettes, and quick-action bars.

The scaling is distance-based — items closest to the cursor grow most, creating a smooth wave effect across the dock. Pure JavaScript, no library needed.
