## Try It — Cook Up Some Prompts

### Recipe 1: The School Helper (10 mins)

Think of something you're studying in school right now that you find a bit confusing.

Fill in your recipe:
- **Context:** I am _____ years old and I'm learning about _____. I already know _____ but I don't understand _____.
- **Instruction:** Please explain _____ to me.
- **Format:** Use a fun analogy or story. Keep it short — about 3–4 sentences. Use simple words.

Combine into one prompt and run it.

**Did it help you understand? Did anything still confuse you?**
If yes — write a follow-up asking about the bit you still don't get.

---

### Recipe 2: The Creative Helper (10 mins)

Think of something creative you'd like to make (a story, poem, song, joke, game idea).

Fill in your recipe:
- **Context:** I want to create _____. It's for _____ (yourself, a friend, school, fun). The main character / topic is _____.
- **Instruction:** _____ (be specific — what should it do/say/feel like?)
- **Format:** _____ (length, style, who wrote it, how it should sound)

Run your recipe prompt. Then try changing just ONE ingredient and run it again. Which version was better?

---

### Recipe 3: The Missing Ingredient Experiment (10 mins)

This one is sneaky — it shows you *why* all three ingredients matter.

Pick a topic (anything you like). Write the prompt THREE times:
1. With ONLY Context (no instruction or format)
2. With ONLY Instruction (no context or format)
3. With ALL THREE ingredients

Compare all three outputs.

| Version | Output quality (1–10) | What was missing? |
|---------|-----------------------|-------------------|
| Context only | | |
| Instruction only | | |
| All three | | |

**What did each missing ingredient affect?**

## Reflect
1. Which ingredient do you find easiest to write? Which is hardest?
2. If you could only pick TWO of the three ingredients, which would you keep?
3. Can you think of a time in real life (not AI) when giving someone more context would have helped?
