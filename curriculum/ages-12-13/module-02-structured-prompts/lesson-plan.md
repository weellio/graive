# Lesson Plan: Your First Structured Prompt
### Builder Level (Ages 12–13) | Module 02

---

## Objective
By the end of this lesson, you'll be able to write a structured prompt using the RCTFC framework that produces a significantly better AI output than an unstructured prompt.

---

## What You'll Need
- Access to a free AI tool (ChatGPT or Claude)
- Your notebook or a Google Doc
- The RCTFC reference card below (screenshot it or print it)
- About 40–50 minutes

---

## The RCTFC Quick Reference Card

```
┌─────────────────────────────────────────────┐
│  R — ROLE        Who should the AI be?       │
│  C — CONTEXT     What's the background?      │
│  T — TASK        What exactly do you need?   │
│  F — FORMAT      How should it present it?   │
│  C — CONSTRAINTS What to avoid/stay within?  │
└─────────────────────────────────────────────┘
```

Screenshot this and keep it open while you work.

## Try It — The Before/After Experiment

### Step 1: Pick Your Topic (5 mins)

Choose ONE thing you actually want help with. Real is better than made-up. Ideas:
- Explaining a school topic you're studying
- Writing a message to someone
- Getting ideas for a creative project
- Understanding something from the news
- Planning something (an event, a project, a video)

Write your topic down: _______________________

---

### Step 2: Write the "Bad" Prompt (5 mins)

Write the simplest, most direct version of your request — just instinct, no structure.

Example: *"Explain the French Revolution"* or *"Help me apologise to my mum"*

Your bad prompt: _______________________

Paste it into the AI. Copy the output into your notebook and label it **Output A**.

---

### Step 3: Build Your RCTFC Prompt (15 mins)

Now build a structured version using the framework. Fill in each section:

| Element | Your Answer |
|---------|------------|
| **R** — Role | The AI should act as a... |
| **C** — Context | The situation is... / This is for... / My background is... |
| **T** — Task | I need it to... specifically... |
| **F** — Format | Present it as... (bullet points / short paragraphs / numbered steps / table) |
| **C** — Constraints | Don't use... / Keep it under... / Avoid... |

Now write it as one connected prompt. You can use all the elements in any order that flows naturally — you don't have to label them.

**Your RCTFC prompt:**

```
[write yours here]
```

Paste it into the AI. Copy the output and label it **Output B**.

---

### Step 4: Compare (10 mins)

Look at Output A vs Output B side by side.

Answer these questions:
1. Which is more useful? Why?
2. Which sounds more like what you actually wanted?
3. What single element of RCTFC made the biggest difference?
4. What would you change in your RCTFC prompt to make it even better?

---

### Step 5: Iterate (5 mins)

Take Output B and improve it with a follow-up message. Don't start over — just respond to the AI with one refinement.

Examples:
- "Make it shorter and more casual"
- "Add an example to the second point"
- "Change the tone — it sounds too formal"
- "Give me a version for someone who already knows the basics"

Copy this final output and label it **Output C**.

**Output C should be the best one. If it's not, try one more iteration.**

## Level Up — Build a Reusable Prompt Template

Create a prompt template you could use again and again for a task you do regularly. Ideas:
- A template for summarising something you've read
- A template for getting feedback on your writing
- A template for brainstorming ideas on any topic
- A template for explaining a concept you're struggling with

Use this structure:

```
You are [ROLE].
I need help with [GENERAL TASK TYPE].
Here is the specific situation: [CONTEXT PLACEHOLDER — you'll fill this in each time].
Please [SPECIFIC ACTION] and present it as [FORMAT].
Avoid [CONSTRAINTS].
```

Test your template on two different topics to make sure it works generally, not just for one thing.

## Reflect
Write 3–5 sentences on each:

1. **The RCTFC element I keep forgetting is ___ because...** What helps you remember to include it?

2. **When I compared Output A and Output B, the most surprising difference was...** What does that tell you about how AI works?

3. **Iteration felt [easy/hard/weird] because...** How do you think you'll use back-and-forth prompting in real life?
