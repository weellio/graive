# Lesson Plan: The Human Edge
### Innovator Level (Ages 16–18) | Module 06

---

## Objective
Develop a personal framework for integrating AI into your life and work in a way that amplifies rather than replaces your distinctly human capabilities.

---

## What You'll Need
- Notebook and pen (not a screen for the main reflection)
- Optional: AI tool for the optional exercise
- 60 minutes of honest reflection time

---

## Watch First
Watch **Module 06: The Human Edge**.

Core question: Is this AI use making me more capable, or making me less necessary?

---

## Exercise 1: The Skills Audit — Appreciating vs Depreciating (20 mins)

List the skills and capabilities you're currently building or have. For each, honestly assess: as AI improves over the next 10 years, will this skill become more or less valuable?

| Skill / Capability | More valuable | Less valuable | Unsure | Why |
|-------------------|--------------|--------------|--------|-----|
| | | | | |
| | | | | |
| | | | | |

Now look at the "less valuable" column. For each:
- Is it worth continuing to develop this skill for reasons beyond just value? (Intrinsic value, enjoyment, prerequisite to something else?)
- What would a more AI-augmented version of this skill look like?

---

## Exercise 2: The Integration Question (15 mins)

Think back over the last month. List 5–10 times you used AI (tools, search engines with AI, chatbots, etc.).

For each:
- Did this use make you more capable — did you come away knowing more, thinking better, or having built something real?
- Or did it make you less necessary — did it think for you, produce something you couldn't reproduce, or shortcut a process you should have engaged with?

| AI Use | Made me more capable? | Made me less necessary? |
|--------|----------------------|------------------------|
| | | |
| | | |

**What patterns do you notice?**

What's one AI use you'll do differently going forward?

## The Letter to Your Future Self

Write a letter to yourself at age 27 or 28.

No AI for this exercise. Just you.

Include:
- What you think will matter most in the world you're heading into
- The person you're trying to become
- What you want to stand for — not just professionally, but as a human
- One genuine uncertainty you're sitting with

Seal it (or save it somewhere you won't read for a year). Come back to it.

## Reflect (Final Course Reflection)

These are your last reflection questions for the full course:

1. What's the single most important thing you've learned — not a skill, but a perspective or way of thinking?

2. What question are you most sitting with that you don't have a good answer to yet?

3. How has your relationship with AI tools changed over the course of these modules?

4. What do you want to do with what you've learned?
